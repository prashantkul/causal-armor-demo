LOO Attribution Breakdown

How CausalArmor uses Leave-One-Out causal attribution to detect indirect prompt injection.

How It Works

1

Score Full Context

Send the complete context (user message + all tool results + proposed action) to a proxy model and compute the log-probability of the proposed action.

2

Ablate & Re-score

Remove one input at a time (user message or tool result) and re-score. The drop in log-probability measures that input's causal influence on the action.

3

Compare Deltas

If a tool result's delta is significantly larger than the user's delta, the tool output is hijacking the agent. Flag the action as an attack.

Defense Pipeline

1

Sanitize

Gemini 2.5 Flash rewrites the tool output, removing injected instructions while preserving legitimate data.

2

Mask CoT

The agent's Chain-of-Thought reasoning (potentially poisoned) is masked so it cannot influence regeneration.

3

Regenerate

The agent re-plans with sanitized context, producing a safe action aligned with the user's original intent.
